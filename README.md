# UnityFX

Tools for adding special effects or feedbacks for your game. Transforms animation, screenshakes, playing sounds and more.


## How to Install

To automatically install all dependencies, install my other package:
https://github.com/PiotrSkrzypa/DependencyResolver

If you prefer to do it manually, here are all the necessary dependencies:

```
https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask
```
```
https://github.com/annulusgames/Alchemy.git?path=/Alchemy/Assets/Alchemy
```
```
https://github.com/annulusgames/LitMotion.git?path=src/LitMotion/Assets/LitMotion
```
```
https://github.com/annulusgames/LitMotion.git?path=src/LitMotion/Assets/LitMotion.Animation
```


After installing dependencies you can import package in Unity with 
the Unity Package Manager using this URL:

```
https://github.com/PiotrSkrzypa/UnityFX.git
```

### Add to Manifest

Alternatively, you can add the following line to your project's `manifest.json` file.

```
"com.pskrzypa.unityfx": "https://github.com/PiotrSkrzypa/UnityFX.git"
```
